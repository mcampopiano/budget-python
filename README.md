# Budget API

## Resources
* Users
* Envelopes
* General expenses
* Budgets
* Deposits
* Recurring expenses
* Payments

### Running the server
In the terminal, run the command 
```javascript
git clone git@github.com:mcampopiano/budget-python.git
cd budget-python
```

### Envelopes
Methods supported:
* GET
* POST
* PUT
* DELETE
